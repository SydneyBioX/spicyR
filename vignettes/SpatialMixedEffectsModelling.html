1 Overview

The following will provide a step-by-step guide on how to use mixed-effects models to analyse multiple image datasets.

2 Mixed-effects modelling

2.2 Test for change in colocalisation between Responders and Non-Responders for a specific pair of cells.

Firstly, we can see if whether one cell type tends to be around another cell type in one condition compared to the other.

spicyTest <- spicy(cells, condition = "condition", subject = "subject", from = "CD8+PD1+PDL1-", to = "CD8-PD1+PDL1+")
## Calculating pairwise spatial associations
## Testing for spatial differences across conditions accounting for multiple images per subject
## boundary (singular) fit: see ?isSingular
spicyTest
## Number of cell type pairs:  1 
## Number of differentially localised cell type pairs: 
## [1] 1
top(spicyTest)

## CD8+PD1+PDL1-_CD8-PD1+PDL1+  16.54962   -10.04735 0.02833181 0.02833181
##                                      from            to
## CD8+PD1+PDL1-_CD8-PD1+PDL1+ CD8+PD1+PDL1- CD8-PD1+PDL1+

As the coefficient is negative, we find that CD8-PD1+PDL1+ cells are more likely to be found around CD8+D1+PDL1- cells in Non-Responders.
